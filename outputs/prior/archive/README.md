# Prior Output Archive

This directory contains legacy reference material for the Stage 2 prior and the folded archive of the completed Phase A sweep.

## Subdirectories

- `stage2_phaseA_multiseed_100epoch/`: folded archive of the completed 15-seed, 100-epoch Phase A sweep
- `legacy_data/`: historical analysis and filtering artifacts produced during earlier project phases
- `legacy_outputs/`: older scratch or comparison outputs retained for traceability

## Interpretation Rule

Use this tree as a legacy reference store, not as the current evidence layer. The current evidence layer lives in the repository-level documentation under [`../../docs/`](../../docs).
