# Prior Evaluation Outputs

This directory is part of the current evidence layer for Stage 2 evaluation outputs.

## Current Evidence

Evaluation artifacts are organized by variant and seed to mirror the training archive.

- `none/`
- `q10/`
- `q20/`
- `q30/`

Within each variant directory, evaluation runs are stored as `seed<k>-<epochs>epoch/reference_seed<sample_seed>/`.

## Legacy Reference

Older evaluation batches may appear under the archive tree when they are retained for traceability. They do not alter the current evidence layer.

## Interpretation Rule

Use this directory for current evaluation evidence and provenance. Historical comparison material belongs under `outputs/prior/archive/`.
