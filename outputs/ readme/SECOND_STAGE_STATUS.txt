Stage 2 Status: Pre-training the Domain Knowledge Diffusion Model
===============================================================

Project scope
-------------
This stage focuses on trajectory-only diffusion prior learning, fully decoupled from the target sensor setup.
The objective is to learn a reusable human-motion prior from public pedestrian trajectory data before any sensor-conditioned input adaptation is introduced.

Current best prior
------------------
The strongest current prior is:
- ddpm_eth_ucy_q20_h128

Reason:
- Expanded public-data pre-training improves prior quality more effectively than merely increasing denoiser width on ETH-only data.
- It substantially improves endpoint displacement matching and progression-related behavior.
- It is the best available initialization for future sensor-conditioned stages.

Current conclusion of Stage 2
-----------------------------
A trajectory-only DDPM prior has been successfully established from public pedestrian trajectory data.
The prior is learnable, but ETH-only training shows clear shrinkage and progression limitations.
Increasing denoiser width alone provides partial gains.
Expanded ETH+UCY public-data pre-training produces the strongest current motion prior.
