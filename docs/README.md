## What Each File Is For

- `prior_stage2.md` explains the official Stage 2 registry-driven prior, the conclusions, the variant-first archive layout, and the public reference figures.
- `stage2_phaseA_multiseed_100epoch_report.md` records the completed 15-seed, multi-seed `100`-epoch screening result and the current mainline interpretation.
- phase-specific working files now live under [`archive/phase_working/`](archive/phase_working)
- reference documents now live under [`archive/reference/`](archive/reference)
- historical or superseded narrative material is grouped under [`archive/`](archive)
- `DOCUMENT_CATALOG.md` classifies the repository documentation into current mainline, historical reference, phase-specific working, and output archive layers.
